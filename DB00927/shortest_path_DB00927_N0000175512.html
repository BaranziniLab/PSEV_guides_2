How to travel from Famotidine to the Methylated Sulfonamides PharmacologicClass:


Currently SPOKE doesn’t contain an edge between Famotidine (DB00927) and Methylated Sulfonamides (N0000175512).
To find paths between DB00927 and N0000175512 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00927']
MATCH (m) where m.identifier IN ['N0000175512']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
