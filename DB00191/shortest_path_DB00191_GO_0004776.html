How to travel from Phentermine to the succinate-CoA ligase (GDP-forming) activity MolecularFunction:


Currently SPOKE doesn’t contain an edge between Phentermine (DB00191) and succinate-CoA ligase (GDP-forming) activity (GO:0004776).
To find paths between DB00191 and GO:0004776 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00191']
MATCH (m) where m.identifier IN ['GO:0004776']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
