How to travel from Sodium Tetradecyl Sulfate to the negative regulation of blood coagulation BiologicalProcess:


Currently SPOKE doesn’t contain an edge between Sodium Tetradecyl Sulfate (DB00464) and negative regulation of blood coagulation (GO:0030195).
To find paths between DB00464 and GO:0030195 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00464']
MATCH (m) where m.identifier IN ['GO:0030195']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
