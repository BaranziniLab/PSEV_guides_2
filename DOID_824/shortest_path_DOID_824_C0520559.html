How to travel from periodontitis to the Numbness of tongue SideEffect:


Currently SPOKE doesn’t contain an edge between periodontitis (DOID:824) and Numbness of tongue (C0520559).
To find paths between DOID:824 and C0520559 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DOID:824']
MATCH (m) where m.identifier IN ['C0520559']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
