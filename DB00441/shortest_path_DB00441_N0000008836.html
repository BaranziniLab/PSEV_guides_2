How to travel from Gemcitabine to the Decreased Prostaglandin Production PharmacologicClass:


Currently SPOKE doesn’t contain an edge between Gemcitabine (DB00441) and Decreased Prostaglandin Production (N0000008836).
To find paths between DB00441 and N0000008836 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00441']
MATCH (m) where m.identifier IN ['N0000008836']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
