How to travel from Tropicamide to the Hyperaemia SideEffect:


Currently SPOKE doesn’t contain an edge between Tropicamide (DB00809) and Hyperaemia (C0020452).
To find paths between DB00809 and C0020452 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00809']
MATCH (m) where m.identifier IN ['C0020452']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
