How to travel from Budesonide to the copper ion transmembrane transport BiologicalProcess:


Currently SPOKE doesn’t contain an edge between Budesonide (DB01222) and copper ion transmembrane transport (GO:0035434).
To find paths between DB01222 and GO:0035434 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB01222']
MATCH (m) where m.identifier IN ['GO:0035434']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
