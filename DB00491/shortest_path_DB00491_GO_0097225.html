How to travel from Miglitol to the sperm midpiece CellularComponent:


Currently SPOKE doesn’t contain an edge between Miglitol (DB00491) and sperm midpiece (GO:0097225).
To find paths between DB00491 and GO:0097225 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00491']
MATCH (m) where m.identifier IN ['GO:0097225']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
