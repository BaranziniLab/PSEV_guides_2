How to travel from Chlorambucil to the miRNA Regulation of DNA Damage Response Pathway:


Currently SPOKE doesn’t contain an edge between Chlorambucil (DB00291) and miRNA Regulation of DNA Damage Response (WP1530_r84694).
To find paths between DB00291 and WP1530_r84694 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00291']
MATCH (m) where m.identifier IN ['WP1530_r84694']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
