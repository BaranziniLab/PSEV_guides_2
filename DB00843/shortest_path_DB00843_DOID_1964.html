How to travel from Donepezil to the fallopian tube cancer Disease:


Currently SPOKE doesn’t contain an edge between Donepezil (DB00843) and fallopian tube cancer (DOID:1964).
To find paths between DB00843 and DOID:1964 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00843']
MATCH (m) where m.identifier IN ['DOID:1964']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
