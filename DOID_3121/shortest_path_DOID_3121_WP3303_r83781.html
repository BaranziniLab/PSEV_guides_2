How to travel from gallbladder cancer to the Rac1/Pak1/p38/MMP-2 pathway Pathway:


Currently SPOKE doesn’t contain an edge between gallbladder cancer (DOID:3121) and Rac1/Pak1/p38/MMP-2 pathway (WP3303_r83781).
To find paths between DOID:3121 and WP3303_r83781 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DOID:3121']
MATCH (m) where m.identifier IN ['WP3303_r83781']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
