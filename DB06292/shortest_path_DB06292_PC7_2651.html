How to travel from Dapagliflozin to the Cobalamin (Cbl, vitamin B12) transport and metabolism Pathway:


Currently SPOKE doesn’t contain an edge between Dapagliflozin (DB06292) and Cobalamin (Cbl, vitamin B12) transport and metabolism (PC7_2651).
To find paths between DB06292 and PC7_2651 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB06292']
MATCH (m) where m.identifier IN ['PC7_2651']
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
