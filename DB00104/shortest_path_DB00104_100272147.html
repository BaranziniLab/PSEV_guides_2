How to travel from Octreotide to the CMC4 Gene:


Currently SPOKE doesn’t contain an edge between Octreotide (DB00104) and CMC4 (100272147).
To find paths between DB00104 and 100272147 click on the Cypher query below and then the play button on the upper right corner of the screen!
If there are no paths (max # of edges=3) between these nodes it will say “(no changes, no records)”.

MATCH (n) where n.identifier IN ['DB00104']
MATCH (m) where m.identifier IN [100272147]
WITH collect(n) as nodes, collect(m) as nodes_2
UNWIND nodes as n
UNWIND nodes_2 as m
MATCH path = allShortestPaths( (n)-[*..3]-(m) )
RETURN path
